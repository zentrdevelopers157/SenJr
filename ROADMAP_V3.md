# ROADMAP V3 🚀

Planned features for the next major release (Version 3) of Senjr:

- **Mentor Dashboard Phase 2**: See incoming connection requests, accept/decline, manage sessions.
- **Student Dashboard Phase 2**: Track active mentors, upcoming sessions, progress notes.
- **Messaging / Chat**: In-app DM between connected mentor-student pairs (using Supabase Realtime).
- **Session Scheduling**: Calendar integration for a simple time-slot picker natively within the app.
- **Reviews & Ratings**: After a session, students rate mentors (1–5 stars + written review).
- **Mentor Verification Badge**: Verified ✓ badge after admin approves a mentor profile.
- **Search & Filter**: Filter mentors by category, availability, rating, and experience level.
- **Notifications**: In-app notification bell + email digests for critical updates.
- **Mobile App**: Evaluate React Native or PWA wrapper for native mobile experience.
